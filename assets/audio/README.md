# Audio Assets

This directory contains the WAV audio samples used by the Davidic Lyre app.

## Expected Files

Each string of the lyre requires one mono WAV sample recorded dry (no baked reverb):

| File            | Note | Frequency  |
|-----------------|------|------------|
| `lyre_D2.wav`   | D2   | 73.42 Hz   |
| `lyre_E2.wav`   | E2   | 82.41 Hz   |
| `lyre_F2.wav`   | F2   | 87.31 Hz   |
| `lyre_G2.wav`   | G2   | 98.00 Hz   |
| `lyre_A2.wav`   | A2   | 110.00 Hz  |
| `lyre_B2.wav`   | B2   | 123.47 Hz  |
| `lyre_C3.wav`   | C3   | 130.81 Hz  |
| `lyre_D3.wav`   | D3   | 146.83 Hz  |

## Specifications

- Format: WAV (uncompressed PCM)
- Channels: Mono
- Sample rate: 44.1 kHz or 48 kHz
- Bit depth: 16-bit or 24-bit
- Character: warm, woody gut-string tone; dry recording with no reverb
- Total budget: keep all audio assets under 50 MB for MVP

Place the WAV files in this directory before building or running the app.
